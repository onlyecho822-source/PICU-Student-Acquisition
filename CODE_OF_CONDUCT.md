# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Positive: welcoming language, respecting viewpoints, accepting criticism, showing empathy.
Unacceptable: trolling, harassment, publishing private info without consent.

## Enforcement

Maintainers will take corrective action. Contact: echonate@echouniverse.org
